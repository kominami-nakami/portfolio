# Works Folder

このフォルダ配下に、実績写真を保存します。

## カテゴリ構成
- EVENTSNAP: イベントスナップ写真
- ARTISTVISUAL: アーティスト写真
- VIDEO: 動画作例（サムネイル画像・関連素材）

## フォルダ命名ルール
- URL構成: works/カテゴリ名/案件名_日付
- 例1: works/EVENTSNAP/CLUB_FOO_20260321
- 例2: works/ARTISTVISUAL/ARTIST_BAR_20260410
- 例3: works/VIDEO/BRAND_BAZ_20260501

## 使い方
1. `works` 配下でカテゴリを選び、案件ごとのフォルダを作成する
2. 各フォルダに写真または動画関連素材を格納する
3. トップページには代表作例1件を反映する
4. 詳細はカテゴリ別ページに追加する
